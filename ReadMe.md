## 📖 Overview

**CyberWebCam** is a professional webcam testing suite designed for security researchers, penetration testers, and IT professionals. It allows authorized testing of webcam functionality, location services, and user awareness in controlled environments.

> ⚠️ **IMPORTANT**: This tool is for EDUCATIONAL and AUTHORIZED testing ONLY. Never use without explicit permission.

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎯 **Multiple Templates** | Festival, YouTube Live, Meeting templates |
| 🔗 **Dual Tunneling** | Cloudflare (free) + Ngrok support |
| 📍 **Location Tracking** | GPS coordinates with accuracy |
| 📸 **Webcam Capture** | Automatic frame capture every 3 seconds |
| 💻 **Cross-Platform** | Linux, macOS, Windows support |
| 📁 **Organized Results** | Timestamped directories for each session |
| 🎨 **Modern UI** | Professional responsive templates |
